Estrutura Básica dos Elementos

A estrutura de um documento HTML é constituido de tags principais

  • html: indica que o documento trata-se de um documento de marcação de texto
  • head: cabeçalho da página (nome que aparece na aba do navegador)
  • body: onde se concentra todo o conteúdo a ser apresetado no na página
  • footer: rodapé da página

Usando Inspetor de Elementos

A ferramenta Inspetor de Elementos nada mais é do que a ferramenta do desenvolvedor, que aparece em todos os navegadores que utilizar (cada um com seu nome próprio, mas todos com a mesma função).

Para acessar essa função é só clicar F12 ou clicar com o botão direito do mouse e clicar em Ferramentas do Desenvolvedor.


Falando sobre Tags

As tags HTML são constituídas de três partes principais: tag de abertura, conteúdo e tag de fechamento (algumas tags HTML não são fechadas). As tags possuem como estruturas principais os sinais < e >, sendo divididas em dois tipos:

  • As que precisam de fechamento: são escritas com início < tag > e < / tag >
  • As que não precisam de fechamento, sendo escritas somente < tag / >

As principais tags HTML para criar uma página HTML incluem as tags estruturais como html, head e body; as tags de conteúdo como h1 a h6, p, img e a; e as tags de formatação como b, i e u.


Atributos Básicos

Os atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Há dois tipos de atributos no HTML:

  • Os globais que são aceitos por todas as tags (exemplo: class, id, lang, style);
  • E algumas outras que você pode conferir na documentação;
  • E os específicos que somente algumas tags possuem, como src, disabled, href e label. A estrutura de um atributo é nome_do_atributo=“valor_do_atributo”.

Os atributos globais HTML básicos incluem os seguintes: accesskey, class, contenteditable, contextmenu, data-*, dir, draggable, dropzone, hidden, id, lang, spellcheck, style, tabindex, title e translate.


Executando uma página HTML

Para executar uma página HTML, você precisa de um navegador web instalado em seu computador. Se você já tem um navegador instalado, basta clicar duas vezes no arquivo HTML e ele será aberto no navegador padrão. Se você não tem um navegador instalado, baixe um dos navegadores mais populares como o Google Chrome, Mozilla Firefox ou Microsoft Edge e instale-o no seu computador. Você também pode usar editores de texto simples como o Bloco de Notas ou o Notepad++ para criar uma página HTML e salvá-la com a extensão .html. Em seguida, clique duas vezes no arquivo HTML para abri-lo em seu navegador padrão.


Textos

Em HTML, o texto é marcado com elementos de texto. O elemento mais básico é o < p >, que define um parágrafo. Outros elementos de texto incluem < h1 > - < h6 > para títulos, < a > para links, < strong > para texto em negrito e < em > para texto em itálico. Você pode usar esses elementos de texto para criar uma página da web com conteúdo estruturado e fácil de ler.


Listas Ordenadas e Não Ordenadas

Em HTML, existem dois tipos de listas:

  • listas ordenadas e
  • listas não ordenadas.

As listas ordenadas são usadas para indicar alguma sequência ou numeração. As tags usadas para criar uma lista ordenada são < ol > e < li >. A tag < ol > é usada para definir a lista e a tag < li > é usada para cada item da lista. Já as listas não ordenadas são usadas para listar itens, sem se preocupar com sua sequência. As tags usadas para criar uma lista não ordenada são < ul > e < li >. A tag < ul > é usada para definir a lista e a tag < li > é usada para cada item da lista.

Links

Em HTML, os links são definidos pela tag < a >. Dentro dessa tag incluímos o atributo href (Hypertext Reference), que é o endereço de destino do link. O link HTML permite inserir hiperlinks em diversos elementos, como textos e imagens. Um link precisa sempre apontar para uma URL (endereço) existente em seu site. Caso contrário, você poderá obter uma mensagem de erro 404.
